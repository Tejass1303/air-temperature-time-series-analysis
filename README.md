# Surface Air Temperature Time Series Forecasting

## Overview
This project analyzes historical surface air temperature data using time series analysis techniques. The main goal is to identify seasonal patterns, trends, and forecast future temperature values.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Trend and seasonality analysis
- Rolling statistics visualization
- Heatmaps and yearly trend plots
- Time series forecasting
- Climate trend interpretation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Scikit-learn

## Project Workflow
1. Import libraries
2. Load and preprocess dataset
3. Perform exploratory data analysis
4. Check stationarity
5. Build forecasting model
6. Evaluate forecasting performance
7. Visualize future predictions

## Key Insights
- Strong yearly seasonal pattern observed.
- Gradual warming trend visible after 2012.
- Monthly cycles dominate the dataset.
- Temperature distribution is approximately normal.
